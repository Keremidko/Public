To setup the project
 - restore the database from the .bak file. (MS SQL Server required)
 - change the database connection string named "RdeConnectionString" in the WebConfig file to connect to your database
 
Information about the project:
 This is my Bachelor work for the Technical University of Varna.
 It's a single page web application for basic UML diagrams. I implemented it using AngularJS/ASp.net WebAPi/Bootstrap & others.
